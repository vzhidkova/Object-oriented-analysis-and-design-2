# Выводы: 

## 1. Уменьшилась связанность 

Форма больше не зависит от конкретных типов документа, она знает только об интерфейсе и репозитории.

## 2. Улучшена расширяемость

Чтобы добавить новый тип документа, нужно:

1) Создать новый класс документа.

2) Создать новую фабрику.

3) Добавить её в словарь.

При этом код формы менять не требуется.

## 3. Повышена поддерживаемость

Логику создания можно изменять внутри фабрики, не затрагивая остальной код.
