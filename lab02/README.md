Пттерн PROXY
Программа реализует графический редактор с возможностью отмены действия. 
Где применяется паттерн: Большой холст представляется набором небольших тайликов, покрывающих все пространство холста. Прии изменении в холсте изменяется не весь массив пикселей, а только те таайлики в которые были подвергнуты изменеиямю за счет чего программа тратит намного меньшл памяти на запоминание изменений на холсте.
